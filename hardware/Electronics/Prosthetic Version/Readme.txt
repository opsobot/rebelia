Electronics Board - Prosthetic Version
======================================
This project was made using Osmond PCB.

The version used was OsmondCocoa 1.2.8 for MacOS.

Link: https://www.osmondpcb.com/

You can use the provided Netlist and Partlist files with any other software, to create your own version.

you can also just use the already exported gerber files and postscript files.
